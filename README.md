# Probable-octo-happiness

A simple joke factory in level db.

## Install
```bash
npm install --save probable-octo-happiness
```
## Why ?

A great beginner in level db.

## Examples

### Add joke.
```js
const jf = require('../src/index');

jf.add('Sample joke'); 
```


### Get Joke

```js
const jf = require('../src/index');

jf.get(key);
```
