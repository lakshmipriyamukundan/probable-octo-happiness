# Probable-octo-happiness

A simple joke factory in level db.

## Install
```bash
npm install --save probable-octo-happiness
```
## Why ?

Helps to learn basic operations in level db and oops. A great starter.

## Examples

### Add joke.
```js
const jf = require('../src/index');

jf.add('Sample joke); 
```


### Get Joke

```js
const jf = require('../src/index');

jf.get(key);
```
